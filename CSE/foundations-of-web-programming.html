HTML
  • HTML => Hyper Text Markup Language
  • has 3 sections
    1. Document Type Definition Statement
    2. Head Section
    3. Body Section
  • DocType (DTD) is case-insensitive
  • Syntax:
  • There are some elements that do not need closing tags. (no containers) Such elements are called "Stand Alone elements".
CSS
  • CSS => Cascading Style Sheet
  • Attribute/Tag name => style
  • 3 ways to use CSS with HTML
    1. Inline CSS
    2. Internal/Embedded CSS
    3. External CSS
  • For CLASS attribute: use '.' notation
    eg; class="my-style" => .my-style {...}
  • For ID attribute: use '#' notation
    eg; id="my-style" => #mystyle {...}
JavaScript
  • scripting language; unrelated to Java
  • can be inserted within head, body or external .js file
  • Tag name => script
